Cheetah is a high-performance animation framework and companion application designed for Apple platforms. The project consists of a Swift-based iOS/macOS application and a React-based web dashboard built with Next.js.

The core functionality focuses on managing and displaying sequenced image assets, such as those found in the bird, bonfire, and butterfly directories. These assets are organized into image sets that represent frames of an animation, allowing for smooth playback and interaction within the Cheetah app.

The repository is divided into two main components. The Cheetah directory contains the Xcode project and Swift source files responsible for the native application experience.

The Website directory contains a modern web stack using TypeScript, Tailwind CSS, and Next.js.
